잔디 스폰서

잔디는 국내 외 18만개 다양한 분야의 기업, 단체와 함께 하고 있습니다.

잔디 주요 기능

대화방

주제별로 대화방을 나누다
여러 주제의 대화가 단일 대화방에서 진행될 경우, 모든 대화를 따라가기 쉽지 않습니다. 팀별, 지점별, 혹은 주제별로 대화방을 깔끔하게 나눠보세요.

파일

생각의 흐름대로 파일을 찾다.
파일을 업로드한 대화방이나 파일 포맷, 혹은 이름 중 하나만 알아도 원하는 파일을 빠르게 찾을 수 있습니다.
잔디에 올린 파일은 유효 기간 없이 언제, 어디서나 다시 확인할 수 있습니다.

관리자 메뉴

메신저에 관리자 기능을 더하다.
멤버별 권한 설정, 강퇴 등 팀 관리에 꼭 필요한 기능을 통해 효율적인 협업 환경을 구축할 수 있습니다.

  • 팀 사용량
    파일 용량, 전송 메시지 수 확인 가능

  • 조직 관리
    조직 관리, 직책 관리, 멤버 관리 기능

  • 다운로드 기록
    멤버 별, 기간 별 다운로드 기록 확인

  • 부가 기능 설정
    게시판 (보드뷰), 조직도, 메시지 수신확인 기능

  • 보안 설정
    모바일 다운로드 제한, 문서 미리보기 워터마크 기능

연동

잔디,외부 서비스를 품다.
트렐로, 지라, 구글 캘린더 등 다양한 서비스를 잔디와 연동해 보세요.
내 업무에 최적화된 커뮤니케이션이 가능해집니다.

보안

자료 보안?
잔디가 책임집니다.
잔디 내 모든 메시지, 파일은 저장 즉시 암호화 처리됩니다. 여기에 세계 최고 수준의 보안을 갖춘 아마존 웹 서비스(AWS)까지! 자료 유출과 손실 걱정에서 벗어나세요.
